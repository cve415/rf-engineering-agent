## Component Deep Dive

### 1. Intelligence Layer (The Brain)
* **Current:** Utilizing structured vetting rubrics in `.md` format to guide manual research.
* **Future:** Integration with **Kimi K2.6** for deep technical discovery. The agent will perform multi-hop reasoning to cross-reference GitHub commits with academic citations.

### 2. Frontend Layer (The Interface)
* **Design Philosophy:** "Industrial Command Center" aesthetic.
* **Tech:** Pure HTML5/CSS3. No heavy frameworks (React/Vue) to ensure maximum speed, portability, and zero-dependency maintenance.
* **Visuals:** Real-time progress bars, tactical "glow" states, and JetBrains Mono typography for high-density data display.

### 3. Deployment & DevOps
* **Continuous Improvement:** Leveraging GitHub for version control of the "Logic Layer."
* **Automation:** Transitioning to **GitHub Actions** to trigger agentic scans and update the ecosystem map automatically on a weekly cron job.
